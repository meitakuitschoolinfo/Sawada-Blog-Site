# 機能一覧表（機能要件一覧）

| システム名 | Blog アプリケーション |
| --- | --- |

本アプリの機能を、機能グループ・機能 ID 単位で整理し、対応する画面・コントローラーと紐づけます。
（機能 ID / 画面 ID は本設計書で付与した管理用 ID です）

## 1. 機能グループ

| 機能グループID | 機能グループ名 | 概要 |
| --- | --- | --- |
| A1 | 会員管理 | ユーザーの登録・ログイン・ログアウト |
| B1 | ブログ管理 | ブログ記事の一覧・登録・編集・画像編集・削除 |
