# 画面遷移図

本アプリの画面遷移を示します（Mermaid 記法）。各ノードは画面、矢印はボタン／リンク／処理結果による遷移を表します。

## 凡例

- ノード内表記: `画面名【画面ID】`
- 矢印ラベル: 操作（ボタン名・リンク名）または処理結果

---

## 1. 全体画面遷移図

```mermaid
flowchart TD
    Login["ログイン【WA101】"]
    Register["ユーザー登録【WA102】"]
    List["ブログ一覧【WB101】"]
    BReg["ブログ登録【WB201】"]
    BRegFix["ブログ登録完了【WB202】"]
    Edit["ブログ編集【WB301】"]
    ImgEdit["ブログ画像編集【WB302】"]
    EditFix["ブログ編集完了【WB303】"]
    DelList["削除候補一覧【WB401】"]
    DelDetail["削除記事詳細【WB402】"]
    DelFix["削除完了【WB403】"]

    Login -- "新規登録はこちら" --> Register
    Register -- "Register（登録）" --> Login
    Login -- "Login（認証成功）" --> List
    Login -- "認証失敗" --> Login

    List -- "Blog Register" --> BReg
    List -- "記事カード選択" --> Edit
    List -- "Delete List" --> DelList
    List -- "Logout" --> Login

    BReg -- "登録成功" --> BRegFix
    BReg -- "登録失敗（重複）" --> BReg
    BRegFix -- "メニューへ戻る" --> List

    Edit -- "変更画面へ" --> ImgEdit
    Edit -- "変更（更新成功）" --> EditFix
    Edit -- "更新失敗" --> Edit
    ImgEdit -- "変更（更新成功）" --> EditFix
    ImgEdit -- "更新失敗" --> ImgEdit
    EditFix -- "メニューへ戻る" --> List

    DelList -- "記事カード選択" --> DelDetail
    DelDetail -- "削除成功" --> DelFix
    DelDetail -- "削除失敗" --> DelList
    DelFix -- "メニューへ戻る" --> List

    %% 共通ヘッダーナビ（WB系の各画面から利用可能）
    BReg -. "Home" .-> List
    Edit -. "Home" .-> List
    ImgEdit -. "Home" .-> List
    DelList -. "Home" .-> List
    DelDetail -. "Home" .-> List
```

> 点線は WB 系画面の共通ヘッダーナビゲーション（Home / Blog Register / Delete List / Logout）による遷移を示します（図の簡潔さのため代表として Home のみ表示）。

## 2. 認証フロー（A 系画面）

```mermaid
flowchart LR
    Login["ログイン【WA101】"]
    Register["ユーザー登録【WA102】"]
    List["ブログ一覧【WB101】"]

    Login -- "新規登録はこちら" --> Register
    Register -- "登録 → redirect" --> Login
    Login -- "認証成功 → redirect" --> List
    Login -- "認証失敗 → redirect" --> Login
```

---

## 3. ブログ操作フロー（B 系画面）

```mermaid
flowchart TD
    List["ブログ一覧【WB101】"]

    subgraph 登録
        BReg["ブログ登録【WB201】"]
        BRegFix["登録完了【WB202】"]
    end
    subgraph 編集
        Edit["ブログ編集【WB301】"]
        ImgEdit["画像編集【WB302】"]
        EditFix["編集完了【WB303】"]
    end
    subgraph 削除
        DelList["削除一覧【WB401】"]
        DelDetail["削除詳細【WB402】"]
        DelFix["削除完了【WB403】"]
    end

    List --> BReg --> BRegFix --> List
    List --> Edit --> EditFix --> List
    Edit --> ImgEdit --> EditFix
    List --> DelList --> DelDetail --> DelFix --> List
```
