## レイヤー構成

本アプリは一般的な 3 層（+ エンティティ）の MVC 構成です。

| レイヤー | パッケージ | 役割 |
| --- | --- | --- |
| プレゼンテーション層 | `controller` | HTTP リクエストの受付、画面遷移、入力の受け渡し |
| ビジネスロジック層 | `service` | 業務処理、判定、トランザクション境界 |
| データアクセス層 | `model.dao` | DB アクセス（JpaRepository 継承インターフェース） |
| エンティティ層 | `model.entity` | テーブルとオブジェクトのマッピング |
| ビュー | `resources/templates` | Thymeleaf による画面描画 |

依存方向: `controller → service → dao → entity`
