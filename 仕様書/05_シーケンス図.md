# シーケンス図

本アプリの主要な処理フローをシーケンス図（Mermaid 記法）で示します。

---

## 1. ユーザー新規登録

```mermaid
sequenceDiagram
    actor User as 利用者
    participant C as UserRegisterController
    participant S as UserService
    participant D as UserDao
    participant DB as DB(users)

    User->>C: POST /user/register/process (userName, email, password)
    C->>S: createAccount(userName, email, password)
    S->>D: findByEmail(email)
    D->>DB: select * from users where email=?
    DB-->>D: UserEntity / null
    D-->>S: 検索結果
    alt メール未登録
        S->>D: save(new UserEntity)
        D->>DB: insert into users
        DB-->>D: 保存結果
        S-->>C: true
    else メール登録済み
        S-->>C: false
    end
    C-->>User: redirect:/user/login
```

> 注: 登録成否に関わらずログイン画面へリダイレクトします。

## 2. ログイン

```mermaid
sequenceDiagram
    actor User as 利用者
    participant C as UserLoginController
    participant S as UserService
    participant D as UserDao
    participant DB as DB(users)
    participant Sess as HttpSession

    User->>C: POST /user/login/process (email, password)
    C->>S: loginAccount(email, password)
    S->>D: findByEmailAndPassword(email, password)
    D->>DB: select * from users where email=? and password=?
    DB-->>D: UserEntity / null
    D-->>S: 検索結果
    S-->>C: UserEntity / null
    alt 認証成功
        C->>Sess: setAttribute("user", userEntity)
        C-->>User: redirect:/user/blog/list
    else 認証失敗
        C-->>User: redirect:/user/login
    end
```

## 3. ブログ一覧表示

```mermaid
sequenceDiagram
    actor User as 利用者
    participant C as BlogListController
    participant Sess as HttpSession
    participant S as BlogService
    participant D as BlogDao
    participant DB as DB(blogs)

    User->>C: GET /user/blog/list
    C->>Sess: getAttribute("user")
    Sess-->>C: UserEntity (userId, userName)
    C->>S: findAllBlogPost(userId)
    alt userId != null
        S->>D: findByUserId(userId)
        D->>DB: select * from blogs where user_id=?
        DB-->>D: List<BlogEntity>
        D-->>S: List<BlogEntity>
        S-->>C: List<BlogEntity>
    else userId == null
        S-->>C: null
    end
    C-->>User: blog-list.html (userName, blogList)
```

## 4. ブログ新規登録

```mermaid
sequenceDiagram
    actor User as 利用者
    participant C as BlogRegisterController
    participant Sess as HttpSession
    participant FS as ファイルシステム(static/blog-img)
    participant S as BlogService
    participant D as BlogDao
    participant DB as DB(blogs)

    User->>C: POST /user/blog/register/process (title, date, category, image, detail)
    C->>Sess: getAttribute("user")
    Sess-->>C: UserEntity (userId)
    C->>FS: Files.copy(画像を保存)
    C->>S: createBlogPost(title, date, fileName, detail, category, userId)
    S->>D: findByBlogTitleAndRegisterDate(title, date)
    D->>DB: select ... where blog_title=? and register_date=?
    DB-->>D: BlogEntity / null
    D-->>S: 検索結果
    alt 重複なし
        S->>D: save(new BlogEntity)
        D->>DB: insert into blogs
        S-->>C: true
        C-->>User: blog-register-fix.html
    else 重複あり
        S-->>C: false
        C-->>User: blog-register.html ("既に登録済みです")
    end
```

## 5. ブログ記事編集（テキスト更新）

```mermaid
sequenceDiagram
    actor User as 利用者
    participant C as BlogEditController
    participant Sess as HttpSession
    participant S as BlogService
    participant D as BlogDao
    participant DB as DB(blogs)

    Note over User,C: 画面表示
    User->>C: GET /user/blog/edit/{blogId}
    C->>S: getBlogPost(blogId)
    S->>D: findByBlogId(blogId)
    D->>DB: select * from blogs where blog_id=?
    DB-->>D: BlogEntity / null
    D-->>S: 検索結果
    S-->>C: BlogEntity / null
    alt 記事あり
        C-->>User: blog-edit.html (blogList)
    else 記事なし
        C-->>User: redirect 一覧へ
    end

    Note over User,C: 更新処理
    User->>C: POST /user/blog/update (title, date, category, detail, blogId)
    C->>Sess: getAttribute("user")
    Sess-->>C: UserEntity (userId)
    C->>S: editBlogPost(title, date, detail, category, userId, blogId)
    S->>D: findByBlogId(blogId)
    D-->>S: BlogEntity
    alt userId != null
        S->>D: save(更新後 BlogEntity)
        D->>DB: update blogs
        S-->>C: true
        C-->>User: blog-edit-fix.html
    else userId == null
        S-->>C: false
        C-->>User: blog-edit.html ("更新に失敗しました")
    end
```

## 6. ブログ画像編集

```mermaid
sequenceDiagram
    actor User as 利用者
    participant C as BlogImageEditController
    participant Sess as HttpSession
    participant FS as ファイルシステム(static/blog-img)
    participant S as BlogService
    participant D as BlogDao
    participant DB as DB(blogs)

    User->>C: POST /user/blog/image/update (image, blogId)
    C->>Sess: getAttribute("user")
    Sess-->>C: UserEntity (userId)
    C->>FS: Files.copy(新画像を保存)
    C->>S: editBlogImage(blogId, fileName, userId)
    S->>D: findByBlogId(blogId)
    D-->>S: BlogEntity
    alt fileName が有効かつ画像が異なる
        S->>D: save(画像名更新)
        D->>DB: update blogs
        S-->>C: true
        C-->>User: blog-edit-fix.html
    else fileName が null または同一画像
        S-->>C: false
        C->>S: getBlogPost(blogId)
        S-->>C: BlogEntity
        C-->>User: blog-img-edit.html ("更新失敗です")
    end
```

## 7. ブログ削除

```mermaid
sequenceDiagram
    actor User as 利用者
    participant C as BlogDeleteController
    participant S as BlogService
    participant D as BlogDao
    participant DB as DB(blogs)

    Note over User,C: 削除詳細表示
    User->>C: GET /user/blog/delete/detail/{blogId}
    C->>S: getBlogPost(blogId)
    S->>D: findByBlogId(blogId)
    D-->>S: BlogEntity
    S-->>C: BlogEntity
    C-->>User: blog-delete-detail.html

    Note over User,C: 削除実行
    User->>C: POST /user/blog/delete (blogId)
    C->>S: deleteBlog(blogId)
    alt blogId != null
        S->>D: deleteByBlogId(blogId)
        D->>DB: delete from blogs where blog_id=?
        S-->>C: true
        C-->>User: blog-delete-fix.html
    else blogId == null
        S-->>C: false
        C-->>User: blog-delete.html ("記事削除に失敗しました")
    end
```
