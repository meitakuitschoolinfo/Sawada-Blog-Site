# 画面レイアウト（ワイヤーフレーム）

| システム名 | Blog アプリケーション |
| --- | --- |

各画面のレイアウトを、HTML テンプレートをもとにした簡易ワイヤーフレーム（ASCII）で示します。
項目の詳細は [07_画面定義書.md](07_画面定義書.md) を参照してください。

---

## WA101 ログイン（login.html）

```
+--------------------------------------------------+
|                                                  |
|                    Welcome                       |
|                                                  |
|        [ UserEmail ____________________ ]        |
|        [ Password  ____________________ ]        |
|              [      Login       ]                |
|              新規登録はこちら                      |
|                                                  |
|         （背景: 桜の花びらアニメーション）          |
+--------------------------------------------------+
```

## WB201 ブログ登録（blog-register.html）

```
+--------------------------------------------------------------+
| 共通ヘッダー                                                  |
+--------------------------------------------------------------+
|  {registerMessage}  （例: 新規記事追加 / 既に登録済みです）     |
|                                                              |
|   タイトル  [ ____________________________ ]                 |
|   日付      [ yyyy-MM-dd ▼ ]                                 |
|   カテゴリ  [ ____________________________ ]                 |
|   画像      [ ファイル選択 ]                                  |
|   詳細      [ ________________________________ ]            |
|             [ (textarea)                       ]            |
|                                                              |
|              [ 登録 ]   [ 戻る ]                              |
+--------------------------------------------------------------+
```

## WB301 ブログ編集（blog-edit.html）

+--------------------------------------------------------------+
|  記事編集                                                     |
