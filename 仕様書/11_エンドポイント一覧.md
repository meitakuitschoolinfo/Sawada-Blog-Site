## 2. リクエストパラメータ詳細

### POST /user/register/process

| パラメータ | 型 | 必須 | 説明 |
| --- | --- | :---: | --- |
| userName | String | ○ | ユーザー名 |
| email | String | ○ | メールアドレス |
| password | String | ○ | パスワード |

### POST /user/login/process

| パラメータ | 型 | 必須 | 説明 |
| --- | --- | :---: | --- |
| email | String | ○ | メールアドレス |
| password | String | ○ | パスワード |

### POST /user/blog/register/process（multipart/form-data）

| パラメータ | 型 | 必須 | 説明 |
| --- | --- | :---: | --- |
| blogTitle | String | ○ | ブログタイトル |
| registerDate | LocalDate (yyyy-MM-dd) | ○ | 登録日 |
| category | String | ○ | カテゴリー |
| blogImage | MultipartFile | ○ | 画像ファイル |
| blogDetail | String | ○ | ブログ詳細 |

### POST /user/blog/update

| パラメータ | 型 | 必須 | 説明 |
| --- | --- | :---: | --- |
| blogTitle | String | ○ | ブログタイトル |
| registerDate | LocalDate (yyyy-MM-dd) | ○ | 登録日 |
| category | String | ○ | カテゴリー |
| blogDetail | String | ○ | ブログ詳細 |
| blogId | Long | ○ | ブログID |

### POST /user/blog/image/update（multipart/form-data）

| パラメータ | 型 | 必須 | 説明 |
| --- | --- | :---: | --- |
| blogImage | MultipartFile | ○ | 新しい画像ファイル |
| blogId | Long | ○ | ブログID |

### POST /user/blog/delete

| パラメータ | 型 | 必須 | 説明 |
| --- | --- | :---: | --- |
| blogId | Long | ○ | 削除対象ブログID |

## 3. 補足

- すべてのブログ系エンドポイント（`/user/blog/**`）は、セッションにログインユーザー（`user`）が存在することを前提とします。
- `@RequestParam` の必須パラメータが欠落、または `registerDate` の日付形式が不正な場合、Spring MVC により HTTP 400（Bad Request）が返されます。
- `{blogId}` はパス変数（`@PathVariable`）です。
